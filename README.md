# Flask TODO App

A simple TODO web app built with Flask, SQLAlchemy, and Bootstrap.  
This project is for learning Flask and practicing web development skills.

## Features
- Add tasks
- View task list
- Delete tasks
- (Upcoming) Mark tasks as done / undone
